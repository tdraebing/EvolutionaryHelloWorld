# Installation

The code is written in Typescript. To build and execute the code you will need [nodejs](https://nodejs.org/en/download/ "NodeJS") version 6.8.1 or higher.
When you have installed nodejs, open your terminal, navigate to the project folder and run:
~~~~
npm install
~~~~
That will install all the dependencies. Those are:
~~~~
chai          \\unit testing framework
mocha         \\unit testing framework
ts-node       \\allows unit tests written in Typescript
typescript    \\to transpile typescript to javascript
typings       \\to install typings for other dependencies
~~~~
And also the typings files needed for some dependencies.
To build and run the code use the following command:
~~~~
npm run start
~~~~
